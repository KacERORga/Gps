🔒 Политика конфиденциальности и использования данных

1. Какие данные собираются

NCPGPS может запрашивать доступ к геолокации для построения маршрутов. Эти данные не покидают ваш браузер и не передаются третьим лицам. История маршрутов сохраняется локально в localStorage — только на вашем устройстве.

2. Использование сторонних сервисов

Навигатор использует открытые API:
- OpenStreetMap (OSM) — картографические данные (только загрузка тайлов)
- OSRM — построение маршрутов
- Nominatim — поиск адресов
- ip-api.com — определение города по IP (если GPS недоступен)
Эти сервисы могут собирать обезличенную информацию в соответствии с их собственными политиками.

3. Хранение данных

Ваши маршруты, настройки и архив хранятся только в браузере (localStorage). Мы не имеем к ним доступа. Вы можете очистить историю в любой момент через инструменты браузера.

4. Файлы cookie и отслеживание

Сайт не использует аналитические cookie, не отслеживает пользователей и не передаёт данные рекламным сетям. Единственное локальное хранилище — для архива маршрутов.

5. Ваши права

Вы можете отказаться от использования геолокации в настройках браузера — навигатор продолжит работать с ручным вводом адресов. Вы можете очистить localStorage через консоль браузера или настройки сайта.

6. Изменения политики

NCPGPS — контент-проект. Политика может меняться, изменения публикуются здесь. Текущая версия от 2026 года.

7. Контакты

Если у вас есть вопросы: support@npcgps.space (решение о прочтении письма может быть не принято).

⚡ Решение о конфиденциальности принято
🍪 NCPGPS использует локальное хранилище для архива маршрутов. Геолокация запрашивается только с вашего разрешения. Мы не передаём ваши данные третьим лицам.
